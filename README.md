## Installation

Creta a `.env.local` file and add your [OpenAI API key](https://beta.openai.com/docs/developer-quickstart/your-api-keys) to it.

```sh
git clone https://github.com/ekaone/whatsappbio.git
cd whatsappbio
pnpm install
pnpm run dev
```
